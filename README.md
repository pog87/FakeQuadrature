# FakeQuadrature

Free Python software related the the paper (cite as): 
Quadrature at fake nodes, preprint 2020, link: ???????????

This software extends fake nodes approach to quadrature formulae

Authors: S. De Marchi, G. Elefante, E. Perracchione, D. Poggiali, 
Universita' di Padova

Last modified: 08/01/20.

The script provides an example for using the fake nodes for mitigating 
both Runge and Gibbs phenomena.
